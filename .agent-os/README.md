# RunSmart Agent OS

This is a lightweight project operating layer. It is not a third-party framework and should stay small.

## Structure
- Root routers: `AGENTS.md`, `CLAUDE.md`, `CODEX.md`
- Workflows: `.agent-os/workflows/`
- Standards: `.agent-os/standards/`
- Templates: `.agent-os/templates/`
- Task memory: `tasks/`
- Product and architecture context: `docs/`

## Rule
Agents should route through the smallest useful set of files for the task. Do not read this whole directory by default.
