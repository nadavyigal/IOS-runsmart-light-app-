# RunSmart — SEO Program

Web SEO is secondary to ASO for RunSmart, but a small set of high-intent pages can drive App Store conversions.

## Tier 1 Pages (build these first)

- `runsmart-vs-runna`
- `ai-running-coach`
- `beginner-5k-plan` (positioned around the product)
- `adaptive-marathon-plan`
- `garmin-running-coach`
- `strava-training-plan`
- `returning-to-running-plan`

Each page is a product-led landing, not a content blog. CTA = App Store.

## Tier 2 Pages (after Tier 1 proves)

- Race-distance pages: 10K, half, marathon (each adaptive)
- "How RunSmart's AI works" explainer
- Use-case pages: "RunSmart for Garmin users", "RunSmart for Strava users"

## Tier 3 (lower priority)

- Programmatic role / event pages — only if a clean data set exists
- Educational content (RPE, easy pace, recovery) — only if needed to support trust

## Internal Linking

- Homepage links to top 3 Tier 1 pages
- Each Tier 1 page links to up to 2 other Tier 1 pages
- All pages link to a single "How it works" page

## Schema

- `SoftwareApplication` for product pages
- `FAQPage` where relevant
- `BreadcrumbList`

## Measurement

- Indexed pages
- Top 10 queries
- Click-through rate to App Store CTA
- App Store install rate from web (UTM)
