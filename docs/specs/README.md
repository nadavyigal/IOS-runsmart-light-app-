# Specs

Approved feature specs live here. Agents should not implement a non-trivial feature until a spec or story exists.

## Expected Flow
1. Product brief.
2. Feature spec.
3. Development stories.
4. Implementation plan.
5. Verified implementation.
6. QA and TestFlight review.

Use `.agent-os/templates/feature-spec-template.md` and `.agent-os/templates/dev-story-template.md`.
