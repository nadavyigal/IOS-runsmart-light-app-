# App Store Readiness Checklist

## Product Page
- [ ] App name and subtitle are final.
- [ ] Description is accurate and not overclaiming AI or health outcomes.
- [ ] Keywords are relevant.
- [ ] Screenshots show real app value.
- [ ] Preview video is optional but accurate if included.

## Compliance
- [ ] Privacy nutrition labels match behavior.
- [ ] Health, fitness, and coaching claims are supportable.
- [ ] No medical diagnosis or unsafe training promises.
- [ ] Terms, privacy policy, and support URL are available.
- [ ] Subscription copy is clear if payments exist.

## Technical
- [ ] Production backend is configured.
- [ ] Crash reporting or diagnostics plan exists.
- [ ] App handles offline and permission-denied states.
- [ ] Test account and reviewer instructions are ready.
- [ ] No placeholder names, icons, screenshots, or debug labels remain.
